A Statistical Analysis of Risk Factors Determining Mortgage Rates

Tech: Python (pandas, statsmodels), Econometrics, Regression Analysis, Hypothesis Testing

Project Summary

This project provides a rigorous econometric analysis of the factors that determine mortgage interest rates in the United States. Using a large loan-level dataset from Freddie Mac, I built and evaluated several Ordinary Least Squares (OLS) regression models to quantify the impact of key borrower and loan characteristics. The analysis confirmed that financial risk indicators—primarily **Credit Score, Debt-to-Income (DTI) ratio, and Loan-to-Value (LTV) ratio**—are statistically significant predictors of interest rates. The project demonstrates a complete analytical workflow, from data cleaning and transformation to advanced regression modeling and interpretation of results.

The Business Problem

For lenders, policymakers, and borrowers, understanding the drivers of mortgage rates is critical for risk management, housing affordability, and financial planning. This project aimed to move beyond general assumptions and use statistical methods to answer a core question: **How much do specific risk factors quantitatively impact the interest rate a borrower receives?** The goal was to build a model that could provide data-driven insights for fairer lending policies, more transparent pricing, and better risk assessment practices.

Key Code Snippets

The analysis centered on building robust regression models to isolate the effect of each variable. The following snippets highlight key stages of the modeling process.

1. Data Transformation for Modeling

To better satisfy the assumptions of linear regression and to interpret coefficients as elasticities, I created log-transformed versions of key variables. This is a standard and important step in econometric modeling.

# Create log-transformed variables for semi-log and log-log models
df['log_Interest_Rate'] = np.log(df['Interest_Rate'])
df['log_Credit_Score'] = np.log(df['Credit_Score'])
df['log_UPB'] = np.log(df['UPB'])

2. Building the OLS Regression Model

The core of the analysis was an Ordinary Least Squares (OLS) regression model, built using Python's `statsmodels` library. The model formula explicitly defines the relationship between the dependent variable (Interest Rate) and the independent risk factors.

import statsmodels.formula.api as smf

# Define and fit the log-log regression model
model_log = smf.ols(formula='log_Interest_Rate ~ log_Credit_Score + DTI + LTV + log_UPB', data=df)
results_log = model_log.fit(cov_type='HC1') # Using robust standard errors

# Print the summary of results
print(results_log.summary())

3. Incorporating State-Level Fixed Effects

To account for unobserved, state-specific factors (like local regulations or economic conditions), I included state-level fixed effects in the model. This advanced technique helps to isolate the impact of the borrower-level variables more accurately.

# Add state dummy variables to the model formula
model_fixed_effects = smf.ols(
    formula='log_Interest_Rate ~ log_Credit_Score + DTI + LTV + log_UPB + C(State)', 
    data=df
)
results_fixed_effects = model_fixed_effects.fit(cov_type='HC1')

print(results_fixed_effects.summary())

Challenges

  • Data Cleaning at Scale: The Freddie Mac dataset is extensive. A key initial challenge was efficiently loading, cleaning, and processing the data, which included handling missing values and ensuring all variables were in the correct format for statistical modeling.
  • Model Specification: A central challenge in any econometric project is choosing the correct functional form for the model. I tested multiple specifications (linear, semi-log, and log-log) to determine which best captured the underlying relationships in the data and provided the most economically sensible interpretations.
  • Avoiding Omitted Variable Bias: Initial models showed that borrower characteristics alone could not explain all the variation in rates. The inclusion of state-level fixed effects was a crucial step to control for geographic differences and reduce the risk of omitted variable bias, leading to more reliable coefficient estimates.
